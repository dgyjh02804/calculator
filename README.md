# 代数表达式计算器 (Algebraic Expression Calculator)

(ai写的)
# 下载exe版本可以直接运行!


一个基于 Python/Tkinter 的代数表达式计算器，支持表达式化简、方程求解、因式分解、分式有理化、对数、高次方程等功能。

## 🚀 运行方式

```bash
cd src
pip install sympy
python run.py
```

## ✨ 功能特性

- **表达式化简**：多项式、根号、绝对值、分式、对数、幂表达式
- **方程求解**：一次~四次方程、分式方程、根式方程、绝对值方程、多元方程、联立方程组
- **因式分解**：多项式因式分解（基于 SymPy）
- **自然常数 `e`**：支持 `e`、`e^x`、`ex²` 等表达式
- **完全平方检测**：`√(a²+4a+4)` → `|a+2|`
- **根号有理化**：`√(2/3)` → `√6/3`、分母共轭有理化
- **精确分数**：自实现 Fraction 类，避免浮点误差
- **测试系统**：79 个内置测试，支持批量运行
- **Debug 模式**：4 级详细度 + 5 档速度，跟踪计算过程
- **日志记录**：自动记录计算历史

## 输入格式

| 类型 | 示例 |
|------|------|
| 表达式化简 | `2x+3x`、`√(4x²+8)`、`(x²-1)/(x-1)`、`(x+1)(ex²-1)/x` |
| 因式分解 | `x²-1`、`x²+5x+6` |
| 方程 | `x²-5=0`、`√(x+1)=2`、`-x³+11x²-10x=70` |
| 方程组 | `x+y=5; x-y=1`（分号分隔） |
| 自然常数 | `e`、`e^x`、`ex²` |
| 运算符 | `+` `-` `*` `/` `^` `√(…)` `|…|` `log(base,arg)` |

## 📁 文件结构

```
src/
├── run.py                     # 启动入口
├── core/
│   ├── base.py                # Fraction 分数类、数学常数
│   ├── expression.py          # 表达式系统（代数项、根号、分式、幂、对数…）
│   ├── algebra_parser.py      # 字符串解析器
│   └── solver.py              # 方程求解、方程组消元
└── gui/
    ├── config.py              # 配置管理、日志、Debug 回调
    ├── widgets.py             # 弹窗控件（日志查看器、测试进度…）
    ├── test_data.py           # 测试用例数据
    └── app.py                 # 主界面
```
