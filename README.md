# 代数表达式计算器 (Algebraic Expression Calculator)

(ai写的)

一个基于 Python/Tkinter 的代数表达式计算器，支持表达式化简、方程求解、因式分解、分式有理化等功能。

# 下载exe版本可以直接运行!

## 功能特性

- **表达式化简**：支持多项式、根号、绝对值、分式的化简
- **方程求解**：一元二次方程、分式方程、根式方程、绝对值方程、多元方程、联立方程组
- **因式分解**：对多项式进行因式分解（基于 SymPy）
- **分式智能化简**：
  - 分子分母因式分解约分（如 `(x²-1)/(x-1)` → `x+1`）
  - 分母共轭有理化（如 `1/(√2+1)` → `-1+√2`）
  - 根号内提取完全平方因子（如 `√(-4c²+8d)` → `2√(-c²+2d)`）
- **精确分数运算**：自实现 Fraction 类，避免浮点误差
- **测试系统**：内置多类别测试用例，支持批量测试
- **Debug 模式**：可调节级别和显示速度，跟踪计算过程
- **日志记录**：自动记录计算历史
- **配置文件**：自动保存用户设置

## 运行方式

```bash
pip install sympy
python algebra_gui.py
```

## 输入格式

| 类型 | 示例 |
|------|------|
| 表达式化简 | `2x+3x`、`√(4x²+8)`、`(x²-1)/(x-1)` |
| 因式分解 | `x²-1`、`x²+5x+6` |
| 方程 | `x²-5=0`、`√(x+1)=2` |
| 方程组 | `x+y=5; x-y=1`（分号分隔） |
| 运算符 | `+` `-` `*` `/` `^` `√(…)` `|…|` |

## 文件结构

| 文件 | 说明 |
|------|------|
| `algebra_gui.py` | 图形界面（Tkinter） |
| `algebra_solver.py` | 求解器（解析、化简、求解、因式分解） |
| `algebra_expression.py` | 表达式类（代数项、根号、分式、有理化） |
| `algebra_base.py` | 基础类（Fraction 分数） |
| `algebra_calculator_config.json` | 用户配置文件（自动生成） |
| `logs/` | 计算日志 |
